Portfolio-ready security tooling

Scan a website and turn raw findings into a readable security snapshot.

This web app wraps your Python scanner in a clean browser workflow for security headers, ports, SSL details, and lightweight exposure checks.

Header analysis Fast port checks SSL visibility Exposure hints

Use a public hostname or full website URL. The app will normalize the target automatically.

What it checks

Core headers, open ports, certificate details, HTTPS behavior, exposed response headers, and a few interesting paths.

How it behaves

Scans run synchronously and can still return partial results if one module fails, including a clear error per section.

Deployment note

Port scanning depends on nmap being installed on the machine running this app.
